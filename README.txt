Para ejecutar la parte de ANTLR, en la consola estar situado en "trunk" e ingresar el siguiente comando:

	> ant run-antlr

Para ejecutar la parte de CUP, en la consola estar situado en "trunk" e ingresar el siguiente comando:

	> ant run-cup

Para limpiar todos los archivos creados con los anteriores comandos, ingresar el siguiente comando:

	> ant clean
